O que é uma imagem digital?

Uma imagem digital é formada por milhares de pequenos pontos coloridos chamados pixels. Quanto mais pixels, maior a resolução e mais nítida a imagem.

Cada pixel tem uma cor definida por valores numéricos. Quando você abre uma foto no computador ou celular, na verdade está vendo uma grade enorme de pixels sendo exibida na tela.

Existem dois grandes tipos de imagem: rasterizadas (feitas de pixels, como fotos) e vetoriais (feitas de fórmulas matemáticas, como logos).

📁 Formatos

Formatos mais comuns na internet

JPG

JPEG / JPG

O formato mais popular para fotografias. Comprime muito bem, gerando arquivos pequenos, mas perde um pouco de qualidade a cada salvamento (compressão com perda).

✓ Fotos ✓ Leve ✗ Sem transparência
PNG

PNG

Ideal para imagens com fundo transparente ou ilustrações com cores sólidas. Mantém qualidade total, mas o arquivo é maior que o JPG.

✓ Transparência ✓ Alta qualidade ✗ Arquivo maior
GIF

GIF

Suporta animações simples e é muito usado em memes e reações nas redes sociais. Limitado a 256 cores, não é bom para fotos reais.

✓ Animado ✓ Leve ✗ Poucas cores
WebP

WebP

Formato moderno criado pelo Google. Combina o melhor do JPG e PNG: arquivos menores com alta qualidade e suporte a transparência. Cada vez mais popular na web.

✓ Muito leve ✓ Transparência ✗ Nem tudo abre
SVG

SVG

Formato vetorial: não perde qualidade em nenhum tamanho! Perfeito para ícones e logos que precisam ser grandes ou pequenos sem distorcer.

✓ Redimensionável ✓ Sem perda ✗ Não serve p/ fotos

💡 Dicas

Como escolher o formato certo?

📸

Salvando uma foto

Use JPG. Mantém boa qualidade com arquivo pequeno, perfeito para fotos do dia a dia.

🎨

Logo com fundo limpo

Use PNG. O fundo transparente deixa a imagem perfeita sobre qualquer cor.

🌀

Imagem animada

Use GIF para animações simples ou WebP para animações com mais qualidade.

🔷

Ícone que cresce

Use SVG. Pode ser colocado em qualquer tamanho sem perder nitidez.

🏆 Atividade Prática

Caça aos Formatos!

Encontre imagens de 5 formatos diferentes na internet

  • 1
    Crie uma pasta no seu computador

    Clique com o botão direito na área de trabalho → "Nova pasta" → Nomeie como "Formatos de Imagem".

  • 2
    Pesquise imagens na internet

    Use o Google Imagens ou qualquer site. Para cada formato, procure algo diferente: uma foto, um logo, um ícone animado...

  • 3
    Salve as imagens na pasta

    Clique com o botão direito na imagem → "Salvar imagem como..." → Escolha a pasta criada. Observe qual é a extensão do arquivo!

  • 4
    Verifique o formato de cada imagem

    Clique com o botão direito no arquivo salvo → "Propriedades" para ver o tipo exato do arquivo.

  • 5
    Marque a lista abaixo conforme for encontrando!

    Use o checklist interativo para acompanhar seu progresso.
