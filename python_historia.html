02 — História

Como o Python surgiu?

Curiosidade: O nome Python não vem da cobra! Guido van Rossum se inspirou no grupo de comédia britânico Monty Python's Flying Circus, que ele adorava.
1980s

🧑‍💻 O Criador

Guido van Rossum, programador holandês, trabalhava no projeto ABC e queria criar algo melhor e mais simples.

1991

🎉 Python 1.0 lançado

Primeira versão oficial. Já tinha: funções, módulos, tratamento de erros e tipos de dados básicos.

2000

🚀 Python 2.0

Grande evolução: melhor suporte a texto, coleta automática de memória e muito mais recursos.

2008

⚡ Python 3.0

Versão atual. Melhorou muita coisa, mas quebrou compatibilidade com Python 2 de propósito, para ficar mais limpa.

Hoje

🏆 Top 3 do mundo

Python é consistentemente uma das linguagens mais populares do planeta, usada por Google, NASA, Netflix e muito mais.

04 — Fundamentos

Os fundamentos mais importantes

Copie no Caderno — Conceitos Essenciais 📓 Anotar
━━━ 5 CONCEITOS FUNDAMENTAIS ━━━

1. VARIÁVEL → guarda um valor na memória
   Ex: nome = "João"

2. TIPO DE DADO → que tipo de valor está guardado
   int → número inteiro (ex: 10)
   float → número decimal (ex: 3.14)
   str → texto (ex: "olá")
   bool → verdadeiro/falso (True / False)

3. FUNÇÃO → bloco de código com um nome
   print() → exibe algo na tela
   input() → pede algo ao usuário

4. INDENTAÇÃO → espaços que organizam o código
   Python usa espaços para definir blocos!

5. COMENTÁRIO → texto ignorado pelo Python
   # isso é um comentário

print("Olá,", nome)
print("Você tem", idade, "anos")

# int() transforma texto em número inteiro
Copie no Caderno — Operadores 📓 Anotar
# Operadores matemáticos
+ → soma      - → subtração
* → multiplicação   / → divisão
** → potência    % → resto da divisão

# Exemplos:
a = 10
b = 3
print(a + b) # 13
print(a * b) # 30
print(a ** 2) # 100
print(a % b) # 1 (resto de 10÷3)
Copie no Caderno — Condição (if/else) 📓 Anotar
# Se... senão...

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

# ATENÇÃO: os 4 espaços (indentação)
# são OBRIGATÓRIOS no Python!
Copie no Caderno — Repetição (for) 📓 Anotar
# Repetindo ações com for

    print("Contagem:", i)

# range(1, 6) → gera: 1, 2, 3, 4, 5
# O número final (6) NÃO é incluído

06 — Praticar Agora

Como executar no OnlineGDB

⚠️ Atenção: Siga os passos na ordem. Não pule etapas!
1

🌐 Acesse o site

Abra o navegador (Chrome, Firefox, Edge...) e acesse o endereço abaixo:

www.onlinegdb.com
2

🐍 Selecione Python

No canto superior direito, clique no menu de linguagem e escolha Python 3.

3

📝 Apague o código existente

Selecione tudo com Ctrl + A e delete. Agora a área está em branco.

4

⌨️ Digite seu código

Escreva o código que o professor pediu. Ex: comece com o Olá Mundo:

print("Olá, mundo!")
5

▶️ Execute clicando em Run

Clique no botão verde ▶ Run no topo. O resultado aparece na parte de baixo, na aba "Console".

6

💬 Se pedir input

Quando o código usar input(), clique na área do console e digite sua resposta + Enter.
