📌 O que é uma logomarca?

Uma logomarca é a identidade visual de uma empresa. É aquilo que faz você reconhecer uma marca rapidamente.

📓 COPIAR NO CADERNO:
Logomarca é a representação visual de uma empresa ou marca, usada para identificação.

🧠 Elementos de uma boa logomarca

  • Simplicidade: fácil de entender
  • Memorável: fácil de lembrar
  • Significado: transmite uma ideia
  • Versátil: funciona em qualquer tamanho
📓 COPIAR NO CADERNO:
Uma boa logomarca deve ser:
  • Simples
  • Fácil de lembrar
  • Ter significado
  • Funcionar em qualquer tamanho

🔥 Exemplos famosos

Nike: transmite movimento e velocidade.

McDonald's: cores que estimulam fome e atenção.

Apple: simplicidade e sofisticação.

📓 COPIAR NO CADERNO:
Logos famosas usam formas simples para transmitir ideias fortes.

🎨 Psicologia das cores

  • 🟡 Amarelo → atenção, alegria
  • 🔵 Azul → confiança

📓 COPIAR NO CADERNO:

🛠️ Criando no Canva

  1. Abrir o Canva
  2. Pesquisar por "Logotipo"
  3. Escolher um modelo simples
  4. Editar nome, cores e ícones
  5. Testar em fundo claro e escuro
📓 COPIAR NO CADERNO:
Passos para criar uma logomarca:
  1. Abrir o Canva
  2. Escolher um modelo
  3. Editar nome, cores e ícones
  4. Testar em diferentes fundos
