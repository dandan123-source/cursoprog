Introdução à Linguagem C

Onde vamos programar?

Vamos usar o site OnlineGDB para escrever e executar nossos códigos.

  1. Acesse o site.
  2. Escolha a linguagem C.
  3. Cole o código.
  4. Clique em Run.

Um Pouco da História

A linguagem C foi criada na década de 1970 por Dennis Ritchie. Ela ficou famosa por ser rápida e eficiente, sendo usada até hoje em sistemas operacionais e softwares importantes.

Estrutura Básica

#include <stdio.h>

int main() {

    return 0;
}

Todo programa em C começa dentro da função main().

printf (Mostrar informações na tela)

#include <stdio.h>

int main() {

    printf("Olá, mundo!");

    return 0;
}

O printf serve para mostrar mensagens na tela.

Variáveis

Variáveis são usadas para guardar valores na memória. 

int idade = 18;
float altura = 1.75;
char letra = 'A';
  • int → número inteiro
  • float → número com decimal
  • char → caractere

Mostrando Variáveis com printf

#include <stdio.h>

int main() {

    int idade = 18;

    printf("Idade: %d", idade);

    return 0;
}

O %d é usado para mostrar números inteiros.

Operações Matemáticas

Podemos fazer cálculos com variáveis. 

#include <stdio.h>

int main() {

    int a = 10;
    int b = 5;

    int soma = a + b;
    int sub = a - b;
    int mult = a * b;
    int div = a / b;

    printf("Soma: %d\n", soma);
    printf("Subtração: %d\n", sub);
    printf("Multiplicação: %d\n", mult);
    printf("Divisão: %d\n", div);

    return 0;
}

Operadores matemáticos:

  • + → soma
  • - → subtração
  • * → multiplicação
  • / → divisão
