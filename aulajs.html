💻 Aula 1 - JavaScript (Primeiros Passos)

1. O que é JavaScript?

JavaScript serve para fazer o site responder ações.

  • Clique em botão
  • Mostrar mensagens
  • Interagir com o usuário

2. Botão com ação

<button onclick="mostrarMensagem()">
Clique aqui
</button>
  • onclick → executa ao clicar
  • () → chama a função

3. Função

function mostrarMensagem() {

}
  • function → cria uma função
  • mostrarMensagem → nome da função
  • () → onde poderiam ir informações
  • { } → código que será executado

4. Variável

let texto = "Olá!";
  • let → cria uma variável
  • " " → texto (sempre entre aspas)

5. Mostrar na tela

document.getElementById("saida").innerHTML = texto;
  • document → a página
  • getElementById → pega um elemento pelo ID
  • innerHTML → coloca conteúdo dentro

6. Código completo (para usar no PHP depois)

<button onclick="mostrarMensagem()">Clique</button>

<div id="saida"></div>

<script>
function mostrarMensagem() {
    let texto = "Funcionou!";
    document.getElementById("saida").innerHTML = texto;
}
</script>

👉 Esse código pode ser colocado dentro de um arquivo .php sem mudar nada!

🧠 Exercício 1

Altere o código para:

  • Mudar a mensagem para: "Olá, aluno!"

🧠 Exercício 2

Crie outro botão que:

  • Chame uma nova função (ex: segundaMensagem())
