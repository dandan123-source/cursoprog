📚 Aula: Login simples em PHP

Objetivo: Criar um login simples e entender como o PHP recebe e verifica dados.

🧩 Passo 1 — Criar o formulário

Crie um arquivo chamado index.php dentro da pasta www.

<!DOCTYPE html>
<html>
<head>
    <title>Login</title>
</head>
<body>

<h2>Login</h2>

<form action="verifica.php" method="POST">

    <label>Usuário:</label><br>
    <input type="text" name="usuario"><br><br>

    <label>Senha:</label><br>
    <input type="password" name="senha"><br><br>

    <button type="submit">Entrar</button>

</form>

</body>
</html>

Explicação:

  • form: cria o formulário
  • action: envia os dados para outro arquivo
  • method="POST": envia os dados de forma oculta
  • name: nome que o PHP vai usar para pegar o valor

🧩 Passo 2 — Criar a verificação

Agora crie outro arquivo chamado verifica.php.

<?php

$usuario = $_POST['usuario'];
$senha = $_POST['senha'];

if ($usuario == "user" && $senha == "1234") {
    echo "<h2>Login correto! Bem-vindo.</h2>";
} else {
    echo "<h2>Usuário ou senha incorretos!</h2>";
}

?>

Explicação:

  • $_POST: pega os dados do formulário
  • $usuario / $senha: armazenam os valores digitados
  • if: verifica a condição
  • &&: significa "E"
  • echo: mostra algo na tela

🧠 Como funciona

  1. Abrir o arquivo index.php
  2. Digitar usuário e senha
  3. Clicar em Entrar
  4. Os dados vão para verifica.php
  5. O PHP verifica
  6. Mostra o resultado

⚠️ Importante

Este exemplo é apenas para aprendizado:

  • Não usa banco de dados
  • Não é seguro
  • Senha está no código

💡 Exercícios

  • Trocar usuário e senha
  • Mostrar o nome digitado:
echo "Olá, " . $usuario;
  • Criar outro usuário com else if
