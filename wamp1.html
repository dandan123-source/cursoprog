Entendendo o WAMP Server

Aprenda como funciona o ambiente para rodar PHP no seu computador

O que é WAMP?

WAMP é um pacote que reúne ferramentas para criar servidores locais no Windows.

  • W - Windows
  • A - Apache (Servidor)
  • M - MySQL (Banco de dados)
  • P - PHP (Linguagem)

Como o WAMP funciona?

Quando você instala o WAMP, ele cria um servidor local no seu computador.

Isso significa que você pode acessar seus projetos pelo navegador usando:

http://localhost

Fluxo de funcionamento

Veja como uma página PHP funciona:

  1. O usuário acessa o navegador
  2. Digita localhost/meusite
  3. O Apache recebe a requisição
  4. Se for PHP, o Apache envia para o PHP processar
  5. O PHP pode acessar o banco MySQL
  6. O resultado é enviado de volta ao navegador

Estrutura de pastas

Os arquivos do seu site ficam dentro da pasta:

www

Exemplo:

C:\wamp64\www\meusite

Exemplo simples em PHP

Crie um arquivo chamado index.php:

<?php
  echo "Olá, mundo!";
?>

Acesse no navegador:

http://localhost/meusite

Dica importante

Se o WAMP não estiver ligado, o site não funcionará.

Verifique se o ícone está verde.
