📚 Aula: IF e ELSE em PHP

Objetivo: Aprender a tomar decisões no PHP usando condições.

🧠 O que é o IF?

O if é usado para executar um código apenas se a condição for verdadeira.

<?php

$idade = 18;

if ($idade >= 18) {
    echo "Você é maior de idade";
}

?>
Se a condição for verdadeira, o código dentro das chaves será executado.

🔁 Usando ELSE

O else executa quando a condição do if é falsa.

<?php

$idade = 16;

if ($idade >= 18) {
    echo "Maior de idade";
} else {
    echo "Menor de idade";
}

?>

🔀 Usando ELSE IF

Serve para testar várias condições.

<?php

$nota = 7;

if ($nota >= 9) {
    echo "Excelente";
} else if ($nota >= 7) {
    echo "Aprovado";
} else {
    echo "Reprovado";
}

?>

⚙️ Operadores de comparação

OperadorSignificado
==Igual
!=Diferente
>Maior
<Menor
>=Maior ou igual
<=Menor ou igual

🔗 Operadores lógicos

OperadorSignificado
&&E (AND)
||OU (OR)
!NÃO (NOT)
<?php

$usuario = "admin";
$senha = "123";

if ($usuario == "admin" && $senha == "123") {
    echo "Login correto";
}

?>

🧠 Resumo

  • if → executa se verdadeiro
  • else → executa se falso
  • else if → múltiplas condições
  • Operadores ajudam nas comparações
