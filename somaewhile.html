5. Estrutura de repetição (while)

Serve para repetir algo várias vezes.

<?php
$i = 1;

while ($i <= 5) {
    echo $i . "<br>";
    $i++;
}
?>

Explicação passo a passo:

  • $i começa valendo 1
  • Enquanto for menor ou igual a 5, ele repete
  • Mostra o valor
  • Soma +1

🔁 Exemplo com texto

<?php
$i = 1;

while ($i <= 3) {
    echo "Olá!<br>";
    $i++;
}
?>
