📸 Aula Completa: Álbum de Recordações

Nesta atividade, você irá criar um site com suas memórias, usando HTML e CSS.

🎯 Objetivo

Criar um álbum com 10 fotos e personalizar o visual do site.

💻 PASSO 1 - Instalar o editor

Baixe um dos programas:
  • VS Code
  • Notepad++
👉 Use o link enviado pelo professor.

📁 PASSO 2 - Abrir o WAMP

1. Abra o WAMP Server
2. Clique no ícone (verde)
3. Vá em www directory

Essa pasta é onde seu site deve ficar.

📁 PASSO 3 - Criar sua pasta

Dentro da pasta www, crie uma pasta com seu nome.
Exemplo: joao_album

📄 PASSO 4 - Criar os arquivos

Dentro da pasta, crie:
  • index.html
  • style.css

🖼️ PASSO 5 - Adicionar imagens

1. Baixe 10 imagens suas (ou da internet)
2. Coloque dentro da pasta do projeto
3. Renomeie para:
  • foto1.jpg
  • foto2.jpg
  • ...
  • foto10.jpg

🌐 PASSO 6 - Entendendo o HTML

Exemplo de código:

<h1>Meu Álbum</h1>
👉 Esse é o título — você pode mudar o texto

<img src="foto1.jpg">
👉 Mostra a imagem — você pode trocar o nome da imagem

<p>Minha lembrança</p>
👉 Texto da legenda — você deve personalizar

🎨 PASSO 7 - Editando o CSS

No arquivo style.css, você pode mudar:
  • Cor do fundo: background-color
  • Tamanho: width
  • Espaço: margin e padding
  • Bordas: border

✨ Exemplos para testar

background-color: lightblue;
border: 2px solid black;
border-radius: 10px;

🧠 Use sua criatividade

  • Crie um tema (escuro, colorido, etc)
  • Mude as cores
  • Deixe o site com sua cara

✅ Entrega

  • 10 fotos
  • Legendas personalizadas
  • CSS modificado

💡 Quanto mais personalizado, melhor sua nota!
