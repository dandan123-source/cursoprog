🔵 Módulo 1

Informática Básica

Antes de usar o computador bem, é importante entender suas peças e como elas se chamam. Veja os conceitos essenciais abaixo:

🖱️

Hardware

São todas as partes físicas do computador — tudo que você pode tocar: teclado, mouse, monitor, processador, memória, HD, etc. Se é sólido e dá pra segurar, é hardware!

💿

Software

São os programas e aplicativos — tudo que não é físico, mas que executa no computador. Exemplos: navegador, Word, jogo, sistema operacional. Software = instruções digitais.

🧠

CPU — Processador

É o "cérebro" do computador. Processa todas as instruções e cálculos. Quanto mais rápido o processador (medido em GHz), mais tarefas ele consegue fazer por segundo.

📋

Memória RAM

É a memória de trabalho. Armazena temporariamente os dados dos programas abertos. Quando você desliga o PC, a RAM é apagada. Mais RAM = mais programas abertos ao mesmo tempo.

💾

Armazenamento: HD e SSD

Guardam seus dados de forma permanente.

• HD (disco rígido): mais lento, mais barato, maior capacidade.
• SSD (estado sólido): muito mais rápido, silencioso e resistente.

⚙️

Sistema Operacional

É o software principal que gerencia tudo no computador — conecta o hardware aos programas. Sem ele o PC não funciona. Exemplos: Windows, Linux, macOS.

🟢 Módulo 2

Sistemas Operacionais

Cada sistema operacional tem sua personalidade. Conheça os principais:

🪟

Windows

O mais popular no mundo. Desenvolvido pela Microsoft. Fácil de usar, ótimo para jogos e trabalho. Versões: Windows 10, Windows 11.

🐧

Linux

Sistema gratuito e de código aberto. Preferido por programadores e servidores. Existem várias versões (distribuições) como Ubuntu, Debian, Mint.

🍎

macOS

Sistema da Apple, exclusivo para computadores Mac. Elegante e seguro, muito usado por designers e criadores de conteúdo.

📱

Android

Sistema para celulares e tablets, desenvolvido pelo Google. Baseado em Linux. É o sistema mobile mais usado no Brasil e no mundo.

🟣 Módulo 3

Arquivos e Pastas

Saber organizar seus arquivos é uma habilidade essencial para usar bem o computador!

📄

O que é um Arquivo?

Um arquivo é uma unidade de dados armazenada no computador. Pode ser um documento de texto, uma foto, um vídeo, um programa… Todo arquivo tem um nome e uma extensão (ex: foto.jpg).

📂

O que é uma Pasta?

Uma pasta (ou diretório) é um espaço para organizar arquivos. Funciona como uma caixinha: você coloca arquivos relacionados dentro dela. Pastas podem conter outras pastas!

🎯

Para que servem?

Arquivos e pastas existem para guardar e organizar informações. Sem pastas, todos os arquivos ficariam misturados — imagina procurar uma foto entre 10.000 arquivos soltos!

🗃️

Boas Práticas de Organização

✅ Use nomes descritivos nos arquivos.
✅ Crie pastas por assunto ou data.
✅ Evite deixar tudo na área de trabalho.
✅ Faça backup regularmente!

💡

Dica: A extensão de um arquivo (a parte depois do ponto, como .docx, .mp3 ou .png) indica o tipo de conteúdo e qual programa deve abri-lo. Nunca apague a extensão!

🟠 Módulo 4

Formatos de Arquivo

Cada tipo de arquivo tem um formato específico. Conheça os mais importantes:

🖼️ Formatos de Imagem

Formato Características Uso Ideal
🔵 JPG / JPEG Compressão com perda de qualidade. Arquivo pequeno. Fotos, redes sociais, sites
🟢 PNG Suporta transparência. Sem perda de qualidade. Logos, ícones, imagens com fundo transparente
🟣 GIF Suporta animações simples. Cores limitadas (256). Memes animados, reações, ícones simples
🟤 BMP Sem compressão. Qualidade total mas arquivo enorme. Edição profissional, impressão
🟡 TIFF Alta qualidade, suporta várias camadas. Arquivo grande. Fotografia profissional, impressão gráfica

🎵 🎬 📄 Outros Formatos

🎵

MP3

Formato de áudio comprimido. Reduz o tamanho do arquivo com pequena perda de qualidade. É o mais popular para músicas.

🎬

MP4

Formato de vídeo muito popular. Boa qualidade com arquivo relativamente pequeno. Usado no YouTube, WhatsApp, etc.

📝

DOCX

Formato de documento de texto do Microsoft Word. Suporta formatação, imagens, tabelas. Muito usado em trabalhos e relatórios.

📕

PDF

Portable Document Format — mantém a formatação original em qualquer dispositivo. Ideal para enviar documentos finais, e-books e contratos.

🟡 Módulo 5

Pendrive Bootável

Entenda o que acontece quando você liga o computador e para que serve um pendrive bootável.

🔌

O que é Boot?

Boot é o processo de inicialização do computador. Quando você liga a máquina, ela executa uma sequência de instruções para carregar o sistema operacional. "Dar boot" = ligar/reiniciar o computador.

🖊️

O que é um Pendrive Bootável?

É um pendrive que contém um sistema operacional completo gravado de forma especial. Permite que o computador inicie (boot) a partir do USB, sem usar o sistema que está no HD/SSD.

🛠️ Como funciona a instalação via Pendrive?

1
Baixar a imagem ISO Você baixa o arquivo do sistema operacional (ex: Windows ou Ubuntu) em formato ISO.
2
Gravar no pendrive Usa uma ferramenta (Rufus, Ventoy…) para gravar a ISO no pendrive de forma bootável.
3
Configurar o Boot Entra na BIOS/UEFI do computador e seleciona o pendrive como primeiro dispositivo de boot.
4
Instalar o sistema O computador inicia pelo pendrive e você segue as instruções de instalação na tela.
⚠️

Atenção: Ao instalar um sistema operacional, os dados do disco podem ser apagados. Sempre faça backup dos seus arquivos importantes antes!

🔴 Módulo 6

Ferramentas para Pendrive Bootável

Existem programas gratuitos que facilitam a criação de pendrives bootáveis. Conheça os principais:

⚡

Rufus

O mais popular para Windows. Simples e rápido. Você escolhe a ISO e o pendrive — em poucos cliques está pronto. Excelente para instalar Windows ou Linux.

📦

Ventoy

Permite múltiplas ISOs em um único pendrive! Você instala o Ventoy no pendrive e depois arrasta os arquivos ISO para dentro — tudo fica listado no menu de boot.

🔀

YUMI

Your Universal Multiboot Installer. Similar ao Ventoy, permite criar um pendrive com vários sistemas. Prático para técnicos de informática.

💿

Arquivo ISO

Um arquivo .iso é uma cópia exata de um CD/DVD (imagem de disco). Contém todos os arquivos necessários para instalar um sistema operacional ou programa. É o "pacote" que você grava no pendrive.

Canva

Plataforma de design gráfico online e gratuita (com versão paga). Permite criar apresentações, cartazes, posts para redes sociais, convites e muito mais, usando templates prontos. Não precisa saber design para usar!

Plataforma de jogos e criação onde qualquer pessoa pode desenvolver e publicar seus próprios jogos usando a linguagem Lua. Muito popular entre jovens — é uma excelente introdução à programação e design de jogos.

🌐 roblox.com

☁️

Google Drive

Serviço de armazenamento na nuvem do Google. Permite salvar arquivos online e acessá-los de qualquer dispositivo. Inclui ferramentas como Docs (texto), Sheets (planilhas) e Slides (apresentações) — similares ao pacote Office. 15 GB gratuitos!

🌐 drive.google.com

Todas essas ferramentas são gratuitas para uso básico e estão disponíveis pelo navegador — sem precisar instalar nada! Basta ter internet e uma conta Google ou criar um cadastro.
