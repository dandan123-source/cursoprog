📝 Aula: Criação de Textos no Computador

💡 O que é um editor de texto?

Um editor de texto é um programa utilizado para escrever, editar e organizar textos no computador. Ele permite não só digitar, mas também formatar o texto, inserir imagens, mudar cores, fontes e deixar o documento mais organizado e profissional.

Esses programas são muito usados na escola, no trabalho e no dia a dia.

Exemplos:

  • Microsoft Word
  • LibreOffice Writer
  • WPS Office
  • Google Docs
  • Canva

📄 Tipos de arquivos de texto

Quando salvamos um documento, ele é guardado em um formato de arquivo. Cada formato tem uma função diferente.

  • .docx → padrão do Word (editável)
  • .odt → usado no LibreOffice (editável)
  • .pdf → formato fechado, difícil de editar
  • .txt → texto simples, sem formatação

O formato PDF é muito usado para entregar trabalhos, pois mantém o formato original.

✍️ Parágrafo

Parágrafo é um conjunto de frases que desenvolvem uma mesma ideia. Ele ajuda a organizar o texto e facilitar a leitura.

Normalmente, iniciamos um parágrafo com recuo e finalizamos quando mudamos de linha.

📏 Espaçamento

O espaçamento controla a distância entre as linhas do texto.

  • 1,0 → mais junto
  • 1,5 → ideal para leitura
  • 2,0 → mais espaçado

Espaçamento é a distância entre as linhas. O mais usado é 1,5.

📐 Alinhamento

O alinhamento define como o texto se posiciona na página.

  • Esquerda → padrão
  • Centro → centralizado
  • Direita → alinhado à direita
  • Justificado → alinhado dos dois lados (mais organizado)

🔤 Formatação de texto

A formatação serve para destacar partes importantes do texto.

  • Negrito → destaque forte
  • Itálico → destaque leve
  • Sublinhado → destaque com linha

↔️ Recuo

O recuo é o espaço no início do parágrafo. Ele ajuda a separar ideias e deixar o texto organizado.

🧪 Atividade

Agora é a sua vez!

  • Abra um editor de texto
  • Crie um texto com no mínimo 20 linhas
  • Use folha A4

Tema:

  • Uma história da sua vida
  • Algo importante para você

Formatação obrigatória:

  • Alinhamento justificado
  • Espaçamento 1,5
  • Parágrafos organizados

Ao final, adicione uma imagem relacionada ao seu texto.
