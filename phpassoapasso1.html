💻 Aula 1 - Introdução ao PHP

Hoje você vai aprender o básico do PHP passo a passo.

1. Primeiro código em PHP

Crie um arquivo chamado index.php:

<?php
echo "Olá, mundo!";
?>

echo serve para mostrar algo na tela.

2. Variáveis

<?php
$nome = "João";
echo $nome;
?>

Variáveis começam com $ e guardam valores.

3. Texto + variável

<?php
$nome = "Maria";
echo "Olá " . $nome;
?>

O ponto (.) junta textos.

4. PHP criando HTML

<?php
echo "<h1>Olá mundo</h1>";
?>

O PHP pode gerar HTML.

5. Formulário

<form>
    <input type="text" name="nome">
    <button>Enviar</button>
</form>

<?php
if ($_GET) {
    echo "Olá " . $_GET['nome'];
}
?>

Digite seu nome e veja o resultado.

Exercício

  • Crie uma variável $cidade
  • Mostre: "Eu moro em ___"

Resumo

  • echo → mostra na tela
  • $ → variável
  • . → junta textos
  • $_GET → pega dados
